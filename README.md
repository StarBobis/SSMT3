# SSMT 3.0系列

面向未来的全新架构与创新设计的新版本SSMT

# 项目说明

随着时代和技术的进步，Mod制作流程需要进行大幅度的变更和优化，这种变更会导致当前现有的流程和经验完全失效，

为此，我将新的技术和新的设计全部放在了3.x版本中，如果你喜欢频繁变动的令人激动的新特性和新设计，欢迎体验3.0系列SSMT。

如果你是SSMT生产力用户，并且使用旧版本SSMT进行成百上千个Mod的维护，请不要轻易尝试3.0版本，其学习成本与操作流程的变更会对现有的流程造成很大影响。

SSMT 2.X版本已进入运维状态，仅进行数据类型更新和随游戏版本更新的导致的逻辑BUG的修复，你仍然可以很好的使用SSMT 2.X版本来进行Mod制作，直到3.0系列发布正式的Release。

# 请勿用于生产力

此3.0系列版本将会发生非常频繁的设计变化和更新，这将会打破你之前所有的工作流程。
